# SAE 1.04 – Site personnel Symfony

Site web d’identité numérique réalisé avec Symfony.

## Objectif
Présenter mon profil, mon CV et mon portfolio dans le cadre de la SAE 1.04.

## Technologies utilisées
- Symfony
- Twig
- HTML / CSS
- Bootstrap / Bootswatch

## Pages du site
- Accueil
- En savoir plus
- CV (PDF intégré + téléchargement)
- Portfolio (intégration Google Drive)

## Lancer le projet en local
```bash
composer install
symfony serve
